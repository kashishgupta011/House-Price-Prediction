🏠 House Price Prediction

This project predicts house prices using Linear Regression with a Scikit-Learn pipeline. It demonstrates data preprocessing, feature scaling, model training, and evaluation using Python.

🚀 Tech Stack

Python

Pandas & NumPy

Matplotlib & Seaborn

Scikit-Learn

Jupyter Notebook

⚙️ Workflow

Data preprocessing

Train-test split

Linear Regression model training

Model evaluation using RMSE & R² score
